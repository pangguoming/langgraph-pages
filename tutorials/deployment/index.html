    部署 🚀¶

    通过 LangGraph 服务器部署 LangGraph 应用程序有两种免费选项

    • 本地: 用于本地测试和开发的部署。
    • 独立容器（精简版）: 独立容器的有限版本，适用于每年节点执行量预计不超过 100 万次，且不需要 Cron 作业和其他企业级功能的部署。独立容器（精简版）部署选项使用 LangSmith API 密钥免费。

    其他部署选项¶

    此外，您可以使用 LangGraph 平台部署到生产环境

    • 云 SaaS: 连接您的 GitHub 仓库并在 LangChain 的云中部署 LangGraph 服务器。我们管理一切。
    • 自托管数据平面(测试版): 从 控制平面 UI 创建部署并将 LangGraph 服务器部署到您的云中。我们管理控制平面。您管理部署。
    • 自托管控制平面(测试版): 从自托管的 控制平面 UI 创建部署并将 LangGraph 服务器部署到您的云中。您管理一切。
    • 独立容器: 随心所欲地部署 LangGraph 服务器 Docker 镜像。

    欲了解更多信息，请参阅部署选项。
