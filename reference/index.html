    参考¶

    欢迎来到 LangGraph 参考文档！这些页面详细介绍了您在使用 LangGraph 构建时将使用的核心接口。每个部分都涵盖了生态系统中的不同方面。

    提示

    如果您刚开始使用，请参阅LangGraph 基础知识，了解主要概念和使用模式的介绍。

    LangGraph¶

    LangGraph 开源库的核心 API。

    • 图：主要图抽象和使用。
    • 函数式 API：用于图的函数式编程接口。
    • Pregel：受 Pregel 启发的计算模型。
    • 检查点：保存和恢复图状态。
    • 存储：存储后端和选项。
    • 缓存：提高性能的缓存机制。
    • 类型：图组件的类型定义。
    • 配置：配置选项。
    • 错误：错误类型和处理。
    • 常量：全局常量。
    • 通道：消息传递和通道。

    预构建组件¶

    用于常见工作流、智能体及其他模式的高级抽象。

    • 智能体：内置智能体模式。
    • 监督器：编排和委托。
    • 群体：多智能体协作。
    • MCP 适配器：与外部系统的集成。

    LangGraph 平台¶

    用于部署和连接 LangGraph 平台的工具。

    • 命令行界面 (CLI)：用于构建和部署 LangGraph 平台应用程序的命令行界面。
    • 服务器 API：LangGraph 服务器的 REST API。
    • SDK (Python)：用于与 LangGraph 服务器实例交互的 Python SDK。
    • SDK (JS/TS)：用于与 LangGraph 服务器实例交互的 JavaScript/TypeScript SDK。
    • RemoteGraph：用于连接 LangGraph 服务器实例的 Pregel 抽象。
    • 环境变量：使用 LangGraph 平台部署时支持的配置变量。
